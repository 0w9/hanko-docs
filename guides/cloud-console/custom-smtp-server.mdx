sidebarTitle: Custom SMTP Server
title: Send Hanko Emails with your own SMTP Server
description: Learn how to set up a custom SMTP server to send Hanko emails through your own SMTP server.
---

<Note>
  Custom SMTP server is available in Hanko Pro and Enterprise plans. Check out our [pricing page](https://hanko.io/pricing) for more information.
</Note>

1. Get the details of your SMTP server from your email provider. You will need the following information. 

   - Host
   - Port
   - Username
   - Password
   - Sender email address

   In case you're using [Resend](https://resend.com), we've got you covered with a [tutorial](https://www.hanko.io/blog/send-hanko-auth-emails-from-your-domain-via-resend) to set up your SMTP server.

2. Login to the [Hanko Cloud](https://cloud.hanko.io) and select your project.

3. Navigate to `Settings > Email`.

4. Enable the Custom SMTP Server option, fill in the details and click `Save`.
