## Import users

If you're migrating from other auth providers to Hanko, you can import your existing users directly to Hanko Cloud Console.

<Note>User IDs should be in UUID format.</Note>

<Steps>
  <Step title="Head over to Hanko Cloud">
    Navigate over to [Hanko Cloud](https://cloud.hanko.io), choose the project
    and click on 'Users' on the left sidebar.
  </Step>
  <Step title="Click on import">
    <Frame>

        style={{ borderRadius: "0.5rem" }}

    </Frame>
  </Step>
  <Step title="Upload the file and import">

## Export users

You can use the [Hanko Admin API](https://docs.hanko.io/api-reference/admin/user-management/get-a-list-of-users) to retrieve a list of users with all their details, for export purposes.

<Note>Hanko Admin API is a paid feature.</Note>
